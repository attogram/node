GUI Wallet is standalone desktop wallet application.

It has also integrated miner

There are  versions for both operating systems:

* [Windows wallet](https://phpcoin.net/download/phpcoin-wallet-win.exe)
* [Linux wallet](https://phpcoin.net/download/phpcoin-wallet-linux)

For manual building Wallet app check [https://github.com/phpcoinn/electron-wallet](https://github.com/phpcoinn/electron-wallet)
